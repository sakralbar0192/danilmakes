## Фаза 2 — Запуск ✅ (архив)

### 2A

- Push в git
- Реальный телефон в `contact.ts`
- Блок «Услуги и цены» на главной
- SEO: Open Graph, `robots.txt`, `sitemap.xml`
- Мобильная вёрстка (базовая)
- Кейсы в портфолио (задача → решение → результат)
- Деплой на VPS (`rsync-deploy.sh`)

### 2B

- DNS `danilmakes.ru` + `www` → `109.71.242.7`
- `.env` на VPS: `DOMAIN`, `CORS_ORIGIN`
- SSL (Let's Encrypt)
- HTTPS nginx + редирект HTTP → HTTPS
- Smoke-тест: сайт, `/api/health`, форма
- Яндекс.Метрика (110107124)

### Операционное (вне кода, по желанию)

- Сайт в [Яндекс.Вебмастер](https://webmaster.yandex.ru) + отправить sitemap
- Google Search Console
- Тестовая заявка через форму — проверить письмо на почте
