# Roadmap: рекламная страница

Цель: сайт-визитка **danilmakes.ru** для поиска небольших заказов в Красноярске.

---

## Фазы

| Фаза | Статус | Суть |
|------|--------|------|
| **1 — MVP** | ✅ Готово | Лендинг, портфолио, контакты, API, деплой на VPS по IP |
| **2 — Запуск** | 🔄 Текущая | Домен, SSL, git push, P0 без блокеров |
| **3 — Конверсия** | ⏳ | Кейсы, цены, SEO, мобилка, аналитика |
| **4 — Рост** | ⏳ | P1–P3 из списка ниже |

**Текущий блокер:** `danilmakes.ru` ещё не резолвится (DNS не дошёл или A-запись не настроена). Прод доступен по **http://109.71.242.7**.

### 2B — после появления DNS (`danilmakes.ru` → `109.71.242.7`)

- [x] Проверить A-запись: `dig +short danilmakes.ru` (и `www`)
- [x] Обновить `.env` на VPS: `DOMAIN=danilmakes.ru`, `CORS_ORIGIN=https://danilmakes.ru`
- [x] SSL: `DOMAIN=danilmakes.ru ./scripts/init-ssl.sh` на сервере
- [x] HTTPS-конфиг nginx (`site.conf` из шаблона, `site.http.conf` отключён)
- [x] Smoke-тест: `https://danilmakes.ru`, `/api/health`, HTTP → HTTPS редирект
- [x] **Яндекс.Метрика** (счётчик 110107124)

### DNS (Timeweb)

| Запись | Тип | Значение |
